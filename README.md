# RRT Simulator

_**Note**_: I don't actively maintain this repository anymore. PRs are more than welcome to help improve it.

## Dependencies
* Qt5
* Eigen

## Compiling
```bash
$ git clone https://github.com/sourishg/rrt-simulator
$ cd rrt-simulator/
$ chmod +x build.sh
$ ./build.sh
```
Run the exectuable as
```
$ ./bin/rrt-test
```
